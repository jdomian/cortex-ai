# cortex-ai examples

Small, runnable scripts that show how to use the cortex-ai API directly.

| File | Shows |
|------|-------|
| `sample-memory-mine.py` | Mining a project directory into the palace |
| `sample-search.py` | Semantic search against the palace |

These are intentionally minimal. Read the source — it's the cleanest documentation.

For the CLI, run `cortex --help` after installing.

For the MCP server (Claude Code integration), see the README at the repo root.
